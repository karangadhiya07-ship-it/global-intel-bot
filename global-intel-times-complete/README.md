# Global Intel Times

Production-ready Vanilla HTML/CSS/JS news portal for Cloudflare Pages.

## Deploy
1. Upload this folder to GitHub.
2. Connect GitHub repo to Cloudflare Pages.
3. Framework preset: None.
4. Build command: leave empty.
5. Output directory: `/`.

## Environment Variables
Add these in Cloudflare Pages > Settings > Environment Variables:

- GNEWS_API_KEY
- NEWS_API_KEY
- YOUTUBE_API_KEY
- FINNHUB_API_KEY

Optional:
- WEATHER_API_KEY
- SPORTS_API_KEY
- COINGECKO_API_KEY

Open-Meteo and CoinGecko free endpoints are already included.
