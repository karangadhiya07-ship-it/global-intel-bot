# Global Intel Times V8.1

Cloudflare Pages + Functions global intelligence platform.

Included:
- 100+ RSS feeds split across four serverless news shards
- 500+ article target with Cloudflare edge caching and fallback feed
- Duplicate removal, category detection, country detection, risk/importance/trending/popularity scoring
- NewsAPI and GNews support through Cloudflare secrets
- Markets, crypto, weather, earthquakes, wildfires, conflicts, country intelligence, search and admin health endpoints
- Premium dark responsive UI with lazy loading and large article grids

Deploy on Cloudflare Pages. Optional secrets: NEWSAPI_KEY, GNEWS_API_KEY, FINNHUB_API_KEY, OPENWEATHER_API_KEY.
